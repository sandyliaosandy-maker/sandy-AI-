## 技术要点
- **占位数据**: 使用 TypeScript 接口定义，便于后续替换为 Contentlayer 类型
- **组件设计**: 遵循单一职责原则，组件可复用
- **样式系统**: 使用 Tailwind CSS 主题配置，保持一致性
- **类型安全**: 所有组件都有完整的类型定义

## 组件说明

### 界面组件
- `Button` - 按钮组件（支持主要、次要、轮廓样式，带图标）
- `Card` - 卡片组件（统一样式，支持链接）
- `Icon` - 图标组件封装（统一使用 lucide-react）

### 内容组件
- `PostCard` - 文章卡片（展示文章信息）
- `PostList` - 文章列表（网格布局）
- `Hero` - 横幅区域（首页大横幅）

### 布局组件
- `Header` - 导航头部（包含 Logo 和导航链接）
- `Footer` - 页脚（版权信息）

## 占位数据
- `mockNews`: 5 条新闻示例数据
- `mockNotes`: 3 条笔记示例数据
