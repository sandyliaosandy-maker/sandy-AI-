# 文档资料目录

本目录存放项目的所有文档和资料文件。

## 目录结构

### 需求文档/
存放项目需求相关文档：
- 功能需求文档
- UI 设计需求文档
- 技术方案文档

### 设计稿/
存放设计相关文件：
- `UI设计稿/` - UI 设计稿
- `原型图/` - 原型图文件
- `参考网站截图/` - 参考网站截图

### 开发文档/
存放开发相关文档：
- API 文档
- 组件文档
- 部署文档
- 开发规范

### 探讨草稿/
存放讨论和方案草稿：
- 技术方案探讨
- 设计思路记录
- 临时想法

## 使用建议

- 需求文档：放在 `需求文档/` 目录
- 设计稿：放在 `设计稿/` 对应子目录
- 开发文档：放在 `开发文档/` 目录
- 草稿和讨论：放在 `探讨草稿/` 目录
