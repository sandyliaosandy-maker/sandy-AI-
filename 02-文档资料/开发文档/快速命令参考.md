## 检查当前目录

如果不确定当前在哪个目录，可以运行：

```bash
pwd
```

应该显示：`/Users/luyu/CascadeProjects/Sandy的AI收藏夹/01-源代码/网站代码`

## 检查文件是否存在

```bash
ls -la package.json
```

如果看到 `package.json` 文件，说明在正确的目录。

## 端口占用检查

如果端口 3002 被占用：

```bash
lsof -ti:3002 | xargs kill -9
```

然后重新启动服务器。
