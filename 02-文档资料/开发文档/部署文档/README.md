## 🎯 快速开始

### 最简单的部署方式（推荐）

1. **运行部署前检查**
   ```bash
   cd "/Users/luyu/CascadeProjects/Sandy的AI收藏夹/01-源代码/网站代码"
   ./部署前检查.sh
   ```

2. **推送到 GitHub**
   ```bash
   git add .
   git commit -m "准备部署"
   git push
   ```

3. **在 Vercel 部署**
   - 访问 https://vercel.com
   - 导入 GitHub 仓库
   - 设置 Root Directory: `01-源代码/网站代码`
   - 点击 Deploy

## 📋 部署准备清单

### 必须准备

- [ ] 代码已通过类型检查 (`npm run type-check`)
- [ ] 代码已通过规范检查 (`npm run lint`)
- [ ] 本地构建成功 (`npm run build`)
- [ ] 代码已推送到 GitHub
- [ ] 内容文件完整（周报、新闻、笔记等）

### 可选准备

- [ ] 配置自定义域名
- [ ] 配置环境变量（如果需要）
- [ ] 配置错误监控（Sentry 等）
- [ ] 配置分析工具（Google Analytics 等）
