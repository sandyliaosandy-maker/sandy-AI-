# Phase 1.1 项目初始化 - 安装说明

## 执行步骤

由于权限限制，请手动执行以下命令来完成项目初始化：

### 1. 安装依赖

```bash
cd "/Users/luyu/CascadeProjects/Sandy的AI收藏夹/01-源代码/网站代码"
npm install
```

### 2. 初始化 Next.js 基础文件

如果 `npm install` 后没有自动创建 Next.js 基础文件，请运行：

```bash
npx create-next-app@latest . --typescript --tailwind --app --no-src-dir --import-alias "@/*" --yes
```

**注意**: 如果提示目录不为空，选择覆盖或手动创建以下文件：
- `app/layout.tsx`
- `app/page.tsx`
- `app/globals.css`

### 3. 配置 Git Hooks（可选）

```bash
npx husky install
npx husky add .husky/pre-commit "npm run check"
npx husky add .husky/commit-msg 'npx --no -- commitlint --edit ${1}'
```

### 4. 创建 Commitlint 配置

创建 `commitlint.config.js`:

```javascript
module.exports = {
  extends: ['@commitlint/config-conventional'],
  rules: {
    'type-enum': [
      2,
      'always',
      [
        'feat',
        'fix',
        'docs',
        'style',
        'refactor',
        'perf',
        'test',
        'chore',
        'content',
      ],
    ],
  },
}
```

### 5. 创建 VS Code 配置（可选）

创建 `.vscode/settings.json`:

```json
{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true,
    "source.organizeImports": true
  },
  "[typescript]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  },
  "[typescriptreact]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  }
}
```

创建 `.vscode/extensions.json`:

```json
{
  "recommendations": [
    "dbaeumer.vscode-eslint",
    "esbenp.prettier-vscode",
    "bradlc.vscode-tailwindcss"
  ]
}
```

### 6. 验证安装

```bash
# 检查依赖
npm list

# 类型检查
npm run type-check

# 代码检查
npm run lint

# 启动开发服务器（使用端口 3002）
npm run dev
```

访问 [http://localhost:3002](http://localhost:3002)
