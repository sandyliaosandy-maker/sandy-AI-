# 用AntiGravity轻松完

**来源**: 少数派

## 金句

💎 我们分析的不仅仅是账单，而是生活的切片。
💎 在算法与消费主义合谋裹挟的时代，数字棱镜折射出真实的习惯。
💎 数据在不经意间记录了你的生活节律。
💎 这个页面的核心价值在于「显影」，将潜伏的行为惯性拉出水面。

## 水下信息

• 使用 Google Gemini 3 Pro 配合 AntiGravity IDE 进行 Vibe Coding 时，其“记忆”和推理能力远超 Cursor 等早期工具，能有效处理涉及10多个文件、数万行代码的复杂项目，而不会像旧工具那样频繁破坏原有逻辑或超出上下文边界。
• 在处理异构数据（如支付宝和微信账单）融合时，一个有效的“水下”策略是：放弃追求底层数据的强行统一与API分类，转而采用“分离上传、并行分析”的架构，并通过视觉与交互层面的优化（如图例分区、联动高亮）来实现逻辑上的统一分析，这能大幅降低工具复杂度和使用门槛。
• 对于微信账单这种导出体验差（最多3个月）、分类信息语焉不详的数据源，直接让大模型（如通过AntiGravity）读取文件结构并询问处理思路，有时它能“自作主张”完成数据切分、自动标签化等绝大部分繁琐工作，在20轮对话内即可实现基本功能，这是绕过传统手动解析壁垒的实用技巧。
• 在消费数据分析中，一个揭示真实消费习惯的“水下”方法是：利用全局筛选功能剥离大额交易后，再观察剩余的小额日常支出（如餐饮、百货），你可能会发现这些看似不起眼的常规消费才是导致“月光”的核心原因，而非偶尔的大额采购。
• 通过“消费关联和弦图”分析分类消费与一周七天的相关性，可以挖掘出个人不易察觉的规律性消费习惯，例如发现“餐饮美食”与“周六/周日”的关联线显著更粗，从而实证“周末是大餐日”的隐性行为模式，这种可视化方法是让数据“讲故事”的关键。

## 案例提取

1. 线上项目 Hessel通过开发并更新“AntiGravity账单分析”工具，为有个人消费分析需求的用户提供服务。该工具支持支付宝和微信账单上传，提供全局交互优化、消费洞察等可视化分析功能。关键更新包括支持微信.xlsx/.csv格式账单、新增消费趋势河流图等实验性图表。作者反馈工具能“显影”隐性消费趋势，成为分析生活切片的“数字棱镜”。

2. 线上项目 Hessel利用Google Gemini的AntiGravity IDE进行Vibe Coding，为个人账单分析项目“AntiGravity”进行开发迭代。核心内容是借助AI辅助编程，高效实现了对异构的微信账单的数据切分、自动标签化及与支付宝账单的混合分析。在不超过20轮对话后即完成了微信账单的集成支持。作者反馈该工具让其找回了对项目的“掌控感”，解决了此前代码膨胀后修改引发的连锁错误问题。
