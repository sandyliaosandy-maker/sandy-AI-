# 20分钟用Claude Cod

**来源**: Peter Yang

## 金句

💎 终端并不神秘，它只是与AI对话的另一种方式。
💎 对于你正在构建的、为了乐趣的东西，跳过权限会让过程有趣得多。
💎 我的七岁孩子还不会打字，但她已经知道如何制作游戏。

## 水下信息

• 使用 `claude --dangerously-skip-permissions` 命令启动 Claude Code，可以绕过每次操作都需要确认权限的步骤，显著提升开发体验和流畅度，这是官方默认启动方式之外的效率技巧。
• 在寻找游戏美术资源时，作者没有完全依赖 Claude 的网页搜索能力，而是预先准备了一个由创作者 Animus 制作的付费像素艺术包（作者支付了5美元），并直接推荐使用，这暗示了在特定领域（如像素游戏美术）存在公认的优质付费资源，比单纯搜索免费资源更高效可靠。
• 作者提到使用名为 Whisper Flow 的语音听写应用，让自己甚至不会打字的7岁孩子也能通过语音与 Claude 交流并参与游戏开发，这揭示了一种降低AI编程工具使用门槛、实现非传统交互（特别是亲子协作）的具体实践方法。
• 教程的核心开发环境是 Cursor 编辑器，而非 Claude Code 官方可能更常关联的终端。作者明确指出选择 Cursor 是因为可以直观地看到 Claude Code 生成的文件，这提供了关于如何更友好地监控和管理AI生成代码的实践见解。
• 整个项目构建流程（设置项目、寻找资源、起草规范、构建最小可行产品并迭代、清理和发布）被压缩在20分钟内完成，这本身传递了一个“水下”信息：利用 Claude Code 等AI编程助手，游戏开发的初始原型构建速度可以远超传统手动编程，尤其适合快速验证想法的业余项目或亲子活动。
