# 特斯拉营收首降，Model S

**来源**: 量子位

## 水下信息

• 特斯拉停产Model S/X的真实原因并非简单的“销量不足”或“情怀不再”，而是为了将核心工厂（弗里蒙特工厂）的宝贵产能和生产线资源，让位给被视为未来核心增长点的Optimus人形机器人，以实现年产100万台的长期目标。这揭示了特斯拉内部资源（尤其是高端制造产能）的激烈竞争和战略优先级排序。
• 特斯拉的资本支出将大幅增加至超过200亿美元，这笔巨额资金的具体投向揭示了其战略重心：新建的六家工厂并非全是汽车工厂，而是包括精炼厂、磷酸铁锂电池工厂、Cybercab工厂、Semi工厂、Optimus工厂和一个新的大型工厂。这暴露了其垂直整合（控制电池原材料、芯片制造）和全面转向AI硬件（机器人、专用车辆）的激进布局。
• 马斯克透露，芯片（特别是内存）供应预计将在三四年后成为公司增长的主要限制。这解释了为什么他此前坚持要建造自己的TerraFab大型晶圆厂。这是一个来自CEO层面的、关于未来供应链关键瓶颈的内部预判，通常不会在公开财报中详细讨论其紧迫性。
• 特斯拉首次披露了FSD（完全自动驾驶）付费用户数为110万，其中70%为预付购买。这一数据属于核心商业机密，揭示了其软件服务的实际渗透率和现金流质量（预付费模式占主导），是评估其AI业务变现能力的关键内幕指标。
• 马斯克在电话会中坦言，人形机器人领域“最大的竞争者来自中国”，并指出“中国以外的人有点低估中国”。这并非客套话，而是基于内部竞争分析和市场情报得出的战略判断，暗示特斯拉在机器人赛道上已感受到来自中国制造业和AI能力的实质性压力，这种来自竞争对手高层的直接评价具有很高的内幕参考价值。

## 案例提取

1. 美国奥斯汀和加州湾区 特斯拉通过部署全无人驾驶Robotaxi车队为公众提供出行服务。在2025年12月启动无安全员测试后，于2026年1月向公众开放了全无人Robotaxi服务。已部署超过500辆Robotaxi。服务已向公众开放，具体用户反馈未在文中明确提及。

2. 全球范围 特斯拉通过提供FSD（完全自动驾驶）软件为特斯拉车主提供高级驾驶辅助服务。FSD V14版本已向车主推送，利用车队收集的每日相当于500年的驾驶数据优化软件。截至2025年末，FSD付费用户数达到110万，同比增长38%。车主反响积极。

3. 美国得州 特斯拉通过建设专用生产线为未来出行市场生产Cybercab（无人驾驶出租车）。该车型无方向盘和踏板，计划于2026年4月启动量产。马斯克预计其产量将比公司所有其他车辆加起来多好几倍。具体用户反馈需待产品上市后获取。

4. 美国弗里蒙特工厂 特斯拉通过改造生产线为市场生产Optimus人形机器人。将原有的Model S和X产线改造为Optimus机器人产线。目标是实现年产100万台Optimus的长期目标。马斯克对产品解决灵巧手、物理AI和量产三大难题有信心。
