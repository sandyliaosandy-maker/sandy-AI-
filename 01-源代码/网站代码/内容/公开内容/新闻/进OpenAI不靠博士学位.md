# 进OpenAI不靠博士学位

**来源**: 量子位

## 金句

💎 博士学位不会自动赋予你好奇心、主动性和研究品味。
💎 尽早进入工业界，因为等博士读完，研究范式可能已换一轮。
💎 你不需要博士学位，只需要真的去做事，并且让别人看得见。
💎 真正稀缺的，是第一眼就能被挑出来的工作，而非“水平还不错”的论文。

## 水下信息

• 进入顶尖AI实验室（如OpenAI、DeepMind、Anthropic）的关键可能不是博士学位，而是**公开、持续地展示自己的研究过程和成果**（例如在GitHub上开源项目、在推特/X上记录实验），这能让你的能力被内部研究员直接看见和衡量，从而获得面试机会。
• 一种有效的“套瓷”策略是：**深入研究目标实验室研究员已发表的论文，并提出具体的改进方案**。这能证明你的认真和能力，并可能获得对方的指导，甚至合作发表论文，这是获得内部推荐和面试机会的绝佳途径。
• 顶尖AI实验室内部可能存在**非正式的“实验性招聘”渠道**，即招聘一些热情和主动性极强但背景非典型的人才，将他们与顶级工程师配对，以探索新的团队组合与潜力，这为背景独特但能力出众者提供了机会。
• 对于本科生或硕士生，进入一线团队的硬通货可能不是“还不错”的顶会论文，而是**具有极高辨识度和冲击力的单项研究成果**（例如能获得NeurIPS最佳论文），这能让你在数千篇论文中立刻被注意到。
• 工业界（尤其是前沿AI实验室）当前可能更稀缺**工程能力极强、能解决真实世界系统问题的人才**，而非单纯的研究型博士。尽早进入工业界接触真实约束，可能比读完一个博士（期间研究范式可能已变）更具职业发展优势。

## 案例提取

1. 美国 OpenAI研究员Keller Jordan通过主动联系论文作者并公开研究（“套瓷研究+推特学术”）为自身职业发展提供服务。他在工作之余联系谷歌研究员Behnam Neyshabur，合作完成一篇入选ICLR 2023的论文，并后续通过公开博客和GitHub项目展示其Muon优化器研究。其工作引起Andrej Karpathy关注，并最终因此进入OpenAI。用户反馈：其研究被圈内大神认可，PR效果显著。

2. 美国 前麦肯锡员工Sholto Douglas通过在GitHub提问和业余时间独立研究为自身职业转型提供服务。他利用晚上和周末时间进行AI研究，并在JAX框架的GitHub仓库积极提问。这引起了谷歌研究员James Bradbury的注意，并获得了面试邀请。用户反馈：他成功进入Google DeepMind，并被内部视为一个关于招聘热情人才的实验。

3. 美国 独立研究者andy jones通过自费租赁算力和发表论文为自身进入AI研究领域提供服务。他独立完成了论文《Scaling Scaling Laws with Board Games》，系统量化了训练与测试计算的权衡。该研究后来推动了OpenAI o1模型的测试时计算范式，并被写入技术报告。用户反馈：他凭借此项研究成为领域焦点，并最终加入Anthropic。

4. 美国 本科生Kevin Wang通过发表高水平学术论文为进入顶尖研究实验室提供服务。他在本科期间完成的研究在NeurIPS 2025的5290篇论文中脱颖而出。该论文《1000 Layer Networks for Self-Supervised RL: Scaling Depth Can Enable New Goal-Reaching Capabilities》最终获得了NeurIPS 2025最佳论文奖。用户反馈：他凭借此成果和导师推荐，在2025年本科毕业后直接加入OpenAI。

5. 美国 OpenAI研究员Noam Brown通过分享非典型研究员经历为潜在AI从业者提供职业路径参考。他在长帖中总结了多位无博士学位研究员通过主动性、公开研究、工程能力和PR进入OpenAI、DeepMind等机构的案例。他提到在AI研究领域，有机会参与重要技术发展且收入可观。用户反馈：他本人从量化交易降薪转做AI研究，并称之为“这辈子做过最好的决定”。
