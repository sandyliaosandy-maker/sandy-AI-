# 域名配置指南

## 🌐 当前状态

你的网站已经部署在 Vercel 上，可以使用 Vercel 提供的免费域名直接访问。

### Vercel 免费域名

- **格式**: `your-project-name.vercel.app`
- **优点**: 
  - ✅ 免费
  - ✅ 自动配置 HTTPS
  - ✅ 全球 CDN 加速
  - ✅ 立即可用
- **缺点**:
  - ❌ 域名较长，不够专业
  - ❌ 包含 `.vercel.app` 后缀

## 📝 是否需要自定义域名？

### 建议使用自定义域名的情况：

1. **面向公众正式发布**
   - 需要更专业的品牌形象
   - 希望域名更易记
   - 需要更好的 SEO 效果

2. **长期运营**
   - 计划长期维护和推广
   - 需要建立品牌认知

### 可以暂时使用免费域名的情况：

1. **测试阶段**
   - 还在开发和测试
   - 需要快速验证功能

2. **个人项目**
   - 主要是个人使用
   - 不需要专业品牌形象

## 🔧 如何配置自定义域名

### 步骤 1: 购买域名

推荐域名注册商：
- **国内**: 阿里云、腾讯云、华为云
- **国外**: Namecheap、GoDaddy、Cloudflare

### 步骤 2: 在 Vercel 添加域名

1. 登录 Vercel Dashboard
2. 选择你的项目
3. 进入 **Settings** → **Domains**
4. 输入你的域名（如 `sandy-ai.com`）
5. 点击 **Add**

### 步骤 3: 配置 DNS

Vercel 会显示需要添加的 DNS 记录：

**方法 A: CNAME 记录（推荐）**
```
类型: CNAME
名称: @ 或 www
值: cname.vercel-dns.com
```

**方法 B: A 记录**
```
类型: A
名称: @
值: 76.76.21.21
```

### 步骤 4: 等待生效

- DNS 生效时间：通常 5 分钟到 24 小时
- Vercel 会自动配置 HTTPS 证书（Let's Encrypt）
- 配置完成后，你的域名就可以访问了

## 💡 小贴士

1. **域名选择**：
   - 简短易记
   - 与品牌相关
   - 避免特殊字符和连字符

2. **DNS 配置**：
   - 使用 CNAME 记录更灵活
   - 配置 www 和根域名（@）
   - 等待 DNS 完全生效

3. **成本**：
   - 域名年费：约 ¥50-100（国内）或 $10-15（国外）
