## 重要：必须在正确的目录下执行命令

所有 npm 命令都必须在 `01-源代码/网站代码/` 目录下执行。

```bash
cd "/Users/luyu/CascadeProjects/Sandy的AI收藏夹/01-源代码/网站代码"
```

## 常用命令

### 开发
```bash
# 启动开发服务器
npm run dev

# 访问 http://localhost:3002
```

### 构建
```bash
# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

### 代码检查
```bash
# 类型检查
npm run type-check

# 代码检查
npm run lint

# 代码格式化
npm run format
```

### 安装依赖
```bash
# 安装所有依赖
npm install

# 安装特定包
npm install <包名> --legacy-peer-deps
```
