# 源代码目录

本目录存放项目的核心源代码文件。

## 目录结构

### 网站代码/
Next.js 项目的源代码，包括：
- `app/` - Next.js App Router 目录（框架要求）
- `components/` - React 组件
- `lib/` - 工具函数库
- `public/` - 静态资源（框架要求）
- 其他 Next.js 配置文件

### 内容源/
Obsidian 知识库的内容文件，包括：
- `收件箱/` - Python 萃取系统的草稿内容
- `公开内容/` - 网站公开内容
  - `新闻/` - 对应 Home 页面（知识萃取内容）
  - `笔记/` - 对应 Notes 页面（小红书/个人写作）
  - `页面/` - 对应 About 等单页
  - `附件/` - 图片和附件（Obsidian 默认附件路径）
- `私有笔记/` - 私有笔记（Git 忽略）

## 注意事项

- `网站代码/` 目录将在初始化 Next.js 项目时创建
- `内容源/` 目录对应 Obsidian 知识库的公开内容部分
- `私有笔记/` 目录不会被 Git 提交
