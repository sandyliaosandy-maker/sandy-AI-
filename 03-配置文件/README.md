# 配置文件目录

本目录存放项目的各种配置文件。

## 目录结构

### 环境配置/
存放环境相关配置：
- `.env.local` - 本地环境变量
- `.env.production` - 生产环境变量
- `.env.development` - 开发环境变量

### Git配置/
存放 Git 相关配置：
- `.gitignore` - Git 忽略文件
- `.gitattributes` - Git 属性配置

### 编辑器配置/
存放编辑器相关配置：
- `.vscode/` - VS Code 配置
- `.idea/` - IntelliJ IDEA 配置

## 注意事项

- 环境配置文件包含敏感信息，不要提交到 Git
- Git 配置文件通常放在项目根目录，这里作为备份或参考
