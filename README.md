# Sandy的AI收藏夹

基于 Obsidian + Git + Next.js 的个人知识库网站项目。

## 项目结构

```
Sandy的AI收藏夹/
├── 01-源代码/          # 核心代码
├── 02-文档资料/        # 文档和资料
├── 03-配置文件/        # 配置文件
├── 04-脚本工具/        # 工具脚本
├── 05-资源文件/        # 资源文件
├── 06-测试文件/        # 测试相关
├── 07-备份存档/        # 备份文件
└── 08-临时文件/        # 临时文件
```

## 快速开始

### 1. 初始化 Next.js 项目

在 `01-源代码/网站代码/` 目录下初始化 Next.js 项目：

```bash
cd 01-源代码/网站代码
npx create-next-app@latest . --typescript --tailwind --app
```

### 2. 配置 Obsidian 知识库

将 Obsidian 知识库的公开内容同步到 `01-源代码/内容源/公开内容/` 目录。

### 3. 安装依赖

```bash
cd 01-源代码/网站代码
npm install
```

### 4. 开发

```bash
npm run dev
```

## 文档

- 需求文档：`02-文档资料/需求文档/`
- 设计稿：`02-文档资料/设计稿/`
- 开发文档：`02-文档资料/开发文档/`
- 技术方案：`02-文档资料/探讨草稿/`

## 注意事项

- 本项目使用中文文件夹命名
- Next.js 的 `app/` 和 `public/` 目录必须使用英文（框架要求）
- `01-源代码/内容源/私有笔记/` 目录不会被 Git 提交

## 技术栈

- **框架**: Next.js 14+ (App Router)
- **内容管理**: Contentlayer
- **样式**: Tailwind CSS
- **知识库**: Obsidian
- **部署**: Vercel
