# 更新日志

## 1.0.0 (2025-01-06)

### 新增功能

- ✅ Markdown 表格解析器
  - 支持标准 Markdown 表格格式
  - 自动识别常见字段（文件路径、日期、标题、标签等）
  - 支持多种日期格式解析

- ✅ 筛选引擎
  - 基于日期的筛选（支持自定义规则）
  - 标签筛选
  - 评分筛选
  - 多条件组合筛选

- ✅ 文件同步功能
  - 保持目录结构
  - 图片附件自动同步
  - 增量同步支持

- ✅ 配置管理
  - JSON 配置文件
  - 路径自动解析
  - 灵活的筛选规则配置

- ✅ 错误处理和日志
  - 详细的错误信息
  - 同步统计信息
  - 友好的控制台输出

### 技术栈

- TypeScript 5.4.5
- Node.js (内置模块)
- date-fns 3.3.1
- fs-extra 11.2.0
